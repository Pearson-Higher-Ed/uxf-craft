{% include '_includes/designSys/atoms/hiddenAnchor' with { 'class': 'ds-animation-trigger' } %}

Embellish

Think of color as an enhancement and not as the primary way to convey meaning. Sighted users benefit by the use of color and contrast to enhance meaning, provide additional emotional impact, or help them feel oriented within consistent color patterns.

Add color last

Besides button transactions, color should be the last thing added to the experience.

Use Pearson palettes

Use Pearson color palettes consistently in designs to ensure an overall unifying experience across Pearson products.

Don't use color for primary meaning

Since color benefits only sighted users, it can never be used as the only way to convey meaning.

Don't cause vibration

Using too many colors at once can cause designs to appear like theyâ€™re vibrating. This can distract and overload users.
